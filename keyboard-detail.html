Keyboard Kingdom

Project Overview: MIDI Rhythm Game â€“ Save the Princess with Piano Skills This project is a rhythm-based music game designed to teach players basic piano skills while delivering a fun and challenging gameplay experience. Players connect a MIDI keyboard and interact with the game by pressing the correct keys as visual cues align with a digital piano interface on screen. Gameplay Mechanics: Real-Time MIDI Input: Players use a physical MIDI keyboard to play notes as falling squares align with keys on a digital piano. Scoring System: Accurate timing earns high scores Missed notes reduce player health Losing all health results in game over and failure to save the princess! Musical Learning: All notes played are real MIDI notes, meaning the music created during gameplay can be played back on a standard piano Players learn actual songs through repetition and rhythm-based feedback
